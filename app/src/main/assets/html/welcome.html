FinanceX Android Welcome Screen

DTN FinanceX for Android offers traders and analysts a broad range of market quotes, charts and news. To get started, please review the following:

Quote List

  • To add new symbols, descriptive labels or expressions to the Quote List, tap the PLUS SIGN ( ) icon. From the menu displayed, chose one of the following:
    • Symbol: Search for a symbol to add to the Quote List by using the symbol tree or by searching for a symbol by name.
    • Label: Add a descriptive label to a set of quotes within your Quote List.
    • Expression: Enter the formula for a spread to add to your Quote List (e.g. @C@1-@C@2).
  • To delete or sort the symbols in the Quote List, tap the PENCIL ( ) icon to enter Edit mode. Tap either the DELETE () icon or the SORT ( ) icon. Tap “DONE” to exit the Edit mode.
  • To view charts and news for a symbol, tap a symbol in the Quote List

Charts

  • To change the Chart interval, tap any of the INTERVAL options ( ) at the top of the chart.
  • To view less/more data within the Chart, use the two-finger pinch and expand feature.
  • To view individual data points on a Chart, touch and hold any data point along the chart. By dragging your finger horizontally along the X-Axis, you can access all data points throughout the chart.
  • To change the chart type (Bar, Line, Candlestick), tap the CHART TYPE icon.

News

  • To view News, tap the NEWS ( ) icon at the bottom center of the screen.
  • To view a category of News, choose from any of the available categories ( ) at the top of the page.
  • To create a keyword News Search, tap the PLUS SIGN ( ) icon at the top right of the page.
  • To edit, delete or sort the News Searches, tap the PENCIL ( ) icon from the top right of the page.
  • Tap the SORT ( ) icon, the EDIT ( ) icon or the DELETE ( ) icon.
  • Additional criteria can be used to group search words:
    • Single quotes are used to indicate the words in the search must be within 50 words of each other and can be in any order.
    • Double quotes are used to indicate the words in the search must be within 2 words of each other and in the order specified.
    • Parenthesis are used to group words together to indicate that all the words in the group should be present, ignoring order and nearness.
  • Related News for a symbol can also be accessed by scrolling down beneath any chart.

Settings

  • To access the app’s settings, tap the SETTINGS ( ) icon at the bottom right of the screen.
    • Auto Refresh Rate: Select the rate and frequency at which your quotes will be updated. You may also tap the REFRESH ( ) icon at the top right of the Quote List page.
    • Account: Select “Disconnect” to disconnect the app from receiving quotes. Select “Connect” to connect the app to receive quotes. Select “Logout” to log out of the app.
    • Help: Select “Welcome” to display this welcome screen. Select “About DTN FinanceX Version: x.x.x to learn more about DTN FinanceX. Select “Email FinanceX Support” to send an email to Customer Service to report a problem.
    • Legal: Select “Privacy Policy” to view DTN’s privacy policy and terms of use. Select “EULA” to view the licensed application end user license agreement.

Google Cloud

  • To utilize the Google Cloud to store your app’s data, you will be prompted with a message upon logging in to the app.

Fonts

  • To change the font size of the text within the FinanceX app, go to your device settings and select DISPLAY and “Screen zoom and font”.
